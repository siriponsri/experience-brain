# Frozen C1 prompt references

Place the exact original Prompt 01 and Prompt 02 files in this directory, then list
their repository-relative paths under `wiki.prompt_references` in `brain.yaml`.

The benchmark records each file's SHA-256 hash in wiki maintenance records, versioned
pages, lessons, and generated context. Do not replace these files after a benchmark run
has started; use a new preregistered prompt version and run identifier instead.
